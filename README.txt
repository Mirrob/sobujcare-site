# SobujCare Website

GitHub-ready website folder for SobujCare.

## Folder Structure
- `index.html`
- `assets/css/style.css`
- `assets/js/app.js`
- `assets/images/*.svg`

## Before Going Live
1. Open `assets/js/app.js`
2. Replace:
   `PASTE_YOUR_GOOGLE_APPS_SCRIPT_WEB_APP_URL_HERE`
   with your Google Apps Script web app URL
3. Update contact details inside `index.html`
4. Replace sample SVG images inside `assets/images/` with your real product photos later

## GitHub Pages
Upload this full folder to your GitHub repo.
If this is your root project, GitHub Pages can publish directly from the main branch root.

## Notes
- Website has BN / EN language toggle
- Live pricing calculator
